# xmfish-crawler
simple web crawler using nodejs

# Install
`mkdir xmfish-crawler`

`cd xmfish-crawler`

`git init`

`git pull https://github.com/adolt/xmfish-crawler.git`

`npm i --save`

# Start
`npm start`

打开浏览器输入 localhost:3000 即可看到爬取的内容
